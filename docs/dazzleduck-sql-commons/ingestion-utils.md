# Ingestion Utilities

> Utilities for managing Arrow streams, dataset handling, and ingestion helpers.

---

## Overview

Provides helpers for:

- Partition paths
- Metadata extraction
- Path validation
- Arrow readers
- Mapping logic

---

## Components

| Class | Purpose |
|------|---------|
| Extractor | Column extraction |
| FileStatus | File metadata |
| TreeAndSize | Scan planning |
| PartitionPruning | Partition selection |
| MappedReader | Reader wrapper |
| ListArrowReader | Dataset reader |

---

## Use Cases

✅ Build ingestion pipelines  
✅ Handle thousands of files  
✅ Track dataset sizes  
✅ Read Arrow efficiently
